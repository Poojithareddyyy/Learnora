# 📚 Learnora - Professional EdTech Learning Platform

A modern, responsive educational technology platform built with React that provides a comprehensive learning experience for students and powerful management tools for administrators.

![Learnora Platform](https://img.shields.io/badge/React-18.2.0-blue) ![Status](https://img.shields.io/badge/Status-Active-success) ![License](https://img.shields.io/badge/License-MIT-green)

## 🌟 Features

### 🎓 For Learners
- **Interactive Tutorial System**: Browse and filter courses by category (Frontend, Backend, Data Science, Design, DevOps)
- **Community Groups**: Join study groups, participate in discussions, and connect with fellow learners
- **Progress Tracking**: Monitor learning journey with detailed course information and ratings
- **Responsive Design**: Seamless experience across desktop, tablet, and mobile devices

### 👨‍💼 For Administrators
- **Comprehensive Dashboard**: Real-time analytics, user management, and platform statistics
- **Course Management**: Create, edit, and manage educational content
- **User Administration**: Manage learners, instructors, and platform permissions
- **Community Moderation**: Monitor discussions and maintain platform quality
- **System Settings**: Configure platform settings and monitor system health

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/CodeArunJ/learnify.git
   cd learnify
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
